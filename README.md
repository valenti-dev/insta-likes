<b>Requirements</b><br>
PHP >= 5<br>
Apache + NGINX<br>
NodeJS + npm<br><br>
<b>Installation</b><br>
Clone git repo<br>
set site base dir /public<br>
Open console and run:
<code>npm i; npm run prod;</code><br>
For WEBP support: remove from NGINX config File Extensions list jpg, png, jpeg<br>
Open site
